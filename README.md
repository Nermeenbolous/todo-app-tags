# todo-app-tags
# To-Do List App with Tagging

A modern, functional To-Do list application built with HTML, CSS, and JavaScript. This project supports task management, categorization through tags, and persistent data storage.

## 🚀 Live Demo
[View the Live App here](https://your-username.github.io/todo-app-tags/)

## ✨ Features
* [cite_start]**Add Tasks**: Create tasks with a required name and an optional tag (School, Work, Personal)[cite: 15, 17, 18].
* [cite_start]**Persistence**: Uses `localStorage` to ensure tasks remain saved even after refreshing the browser[cite: 1, 27].
* **Filtering**: 
    * [cite_start]Filter by status: All, Active, or Completed [cite: 32-35].
    * [cite_start]Filter by category: Dropdown menu to view tasks by specific tags[cite: 36].
* [cite_start]**Validation**: Prevents empty tasks with a real-time error message[cite: 11, 12].
* [cite_start]**Task Counter**: Displays the total number of active tasks remaining[cite: 48, 49].
* [cite_start]**Management**: Mark tasks as complete or delete them permanently[cite: 52, 56].
